Breakthrough

The VAHH is a very advanced VA and would require a number of technological breakthroughs. The breakthroughs that it would require include Ethical understanding. Adaptive learning, and Problem Solving.

Ethical implementation of VAs is crucial to whether they will ever be developed, thus the ethical problems they face must be solved. There are several ethical principles that VAs need to follow, which include making them follow transparency, justice and fairness, non-maleficence, responsibility and intergrity, and privacy. Violations in any of these ethicial principles could interface with the VAHH ever jbeing viable, so it is important to understand what needs to be interfere with the VAHH ever being viable, so it is important to understand what needs to be done to fix these problems.

In order for the VA to be able to interact with people on a daily basis it needs to be able to learn. If we integrate adaptive learning into the VA it could learn from its conversations with the home owner and change how it acts to better suit the inhabitants. In order to do this it needs to learn mappings between verbal commands and basic actions. After it learns this it can piece together verbal commands and understand more complex interactions.

As of now, VAs cannot do anything without already knowing how. If the Ai of the VA could use problem sloving skills and make educated decisions its capabilities would be vastly improvoed. Currently we have the technology to make an AI that is better than us at simple things like a specific task. But, that same AI would have no skill at a different task even if it was only slightly different. This is because it is only coded to interact with that specific task, and when put in a seperate one it would try to perform the same actions and would ultimately fail.

To make the VAHH function and connect would require a specialized system that works specifically for the VA inside. This sort of system already exists on a small scale in simple VA techs such as Amazon Alexa or Apple Siri which can be programmed to control other smart devices. You would need speakers and microphones throughout the house to communicate with the VA, and you would need to connect the VA to any appliances that you want the VA to be able to control.

The infrastructure breakthrough would be the step needed to start building a VAHH. All the components already exist, though the development of said homes will be expensive, certainly more than most homes cost today. For this technology, several market costs need to be reduced or stabilzed.

To investigate how this system could work, we designed a potential experiment to test how this technology would work in the VAHH and how to use it. First, the development company or organization that begins the project should start with a single room connected to the VA. This room should have everything the proposed homes of the future need. This alone may be expensive, but it makes the first step in determining how the construction would work.

If this development works, the company could survey the people who visit the project and get feedback. Using this feedback, they can develop a full VAHH using the knowledge and developments from the single-room experiment. This house could be rented or even just shown as an open house to many people to upscale the project in a realistic way. This experiment would prove that the technology is possible, and it would show that people are willing to invest in these homes, which would make this development a reality.
